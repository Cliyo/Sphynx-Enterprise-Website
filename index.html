Segurança Barata
& Eficiente para você

A Cliyo é uma startup inovadora que foi criada para fornecer soluções de segurança residencial acessíveis e eficazes. Por acaso já passou por uma situação em que se sentiu inseguro em sua própria casa? Mas por conta do alto custo não conseguiu um sistema eficaz? O Sphynx é a solução para você!

Sobre o Sphynx

Como já foi dito anteriormente, o Sphynx é um sistema de segurança eficaz e custo-benefício

Tecnologias

O Sphynx possui o módulo físico equipado com ótimos sensores para liberação da tranca magnética, além do módulo web para controle de fluxo

Custo

Como missão do Sphynx temos o barateamento, por conta disso usamos sensores de baixo custo no produto final para reduzirmos o custo de produção

Locais

Por conta de seu módulo Web para controle de fluxo, o Sphynx pode além de em residências ser usado em escolas ou hospitais para controle de entrada e saída
